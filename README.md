# **SKILIKET DASHBOARD**

## Install and run

### Cloning the repository

Clone the repository:

* HTTPS:

```bash
git clone https://github.com/andresaugom/Skiliket-Dashboard.git
cd Skiliket-Dashboard
```

* SSH: 

```bash
git clone git@github.com:andresgomez31/Skiliket-Dashboard.git
cd Skiliket-Dashboard
```

### Installing dependencies

If you don't have __npm__ (node package manager), install it:

```bash
sudo pacman -S npm
```

Then:

```bash
npm install
```

### Running the project locally

Run:

```bash
npm run dev
```

And then go to the following [link](https://localhost:3000).
